## 1. 项目背景

### 研究问题

乙醇（酒精）如何影响果蝇的运动行为和空间选择偏好？

### 实验概述

将果蝇置于 **T 型迷宫** 中，通过 **DeepLabCut (DLC)** 无标记姿态估计技术追踪其运动轨迹，分析不同浓度乙醇暴露下果蝇在迷宫左臂/右臂的停留偏好。

### 技术路线

```
视频录制 → DLC 姿态追踪 → H5 数据文件 → Python 分析脚本 → 统计图表
```

## 2. 实验设计

### 2.1 T 型迷宫结构

```
        LEFT ARM              HESITATION             RIGHT ARM
        X < 500              500 ≤ X ≤ 700           X > 700
    ┌──────────────┬─────────────────────┬──────────────┐
    │              │                     │              │
    │   左臂       │     犹豫区          │    右臂      │
    │              │   (T型交汇处)        │              │
    └──────────────┴─────────────────────┴──────────────┘
                          ↑ 竖杆 (stem)
                          │ 果蝇由此进入迷宫
```

- 果蝇从底部竖杆进入，到达 T 型交汇处后选择左臂或右臂
- 迷宫被划分为三个区域：**左臂**（X<500）、**犹豫区**（500≤X≤700，即交汇处）、**右臂**（X>700）
- 像素坐标：X 范围 0~1200，Y 范围 0~1000

### 2.2 实验分组

| 分组 | 乙醇浓度 | 个体数 | 说明 |
|------|:------:|:----:|------|
| Air (Ctrl) | 0% | 5 | 空气对照组 |
| 5% EtOH | 5% | 3 | 低浓度 |
| 10% EtOH | 10% | 1 | 中低浓度 |
| 15% EtOH | 15% | 1 | 中高浓度 |
| 20% EtOH | 20% | 2 | 高浓度（视频需校正，见下文） |
| 25% EtOH | 25% | 1 | 最高浓度（视频需校正） |

> **视频偏移校正**：20% 和 25% EtOH 的原始视频存在约 100 像素的右偏，分析时自动将 X 坐标左移 100 像素。新增的 5% 组第三只个体（`5%ETOH-air-WT -3DLC`）同样应用此校正。

### 2.3 DLC 追踪部位

每帧追踪果蝇三个身体部位：

| 部位 | 缩写 | 说明 |
|------|:----:|------|
| 头部 | head | 果蝇头部位置 |
| 腹部 | body | 果蝇腹部/胸部位置 |
| 尾部 | tail | 果蝇尾部位置 |

每个部位每帧输出三个值：`(x 坐标, y 坐标, 置信度)`。置信度范围 0~1，< 0.6 视为追踪失败。

## 3. 数据说明

### 3.1 H5 文件格式

每个 `.h5` 文件 = 一只果蝇在一次实验中的完整追踪数据，由 DeepLabCut 生成。

- **内部路径**：`df_with_missing/table`
- **每行格式**：`[帧号, [head_x, head_y, head_lik, body_x, body_y, body_lik, tail_x, tail_y, tail_lik]]`

### 3.2 文件命名规则

```
<乙醇浓度>_<条件>_<个体ID>_Resnet50_T_trapJul8shuffle1_snapshot_best-30.h5
```

示例：
- `10%ETOH-air-WTDLC_...h5` → 10% 乙醇、空气条件、WT 品系
- `25%EtOH_air_WTDLC_...h5` → 25% 乙醇、空气条件、WT 品系
- `AIR-air-WT2DLC_...h5` → 对照组、WT2 个体

### 3.3 位置计算

```python
if 头、腹、尾均有效 (likelihood > 0.6):
    X = ((head_x + tail_x)/2 + body_x) / 2
    Y = ((head_y + tail_y)/2 + body_y) / 2
elif 仅腹部有效:
    X, Y = body_x, body_y
else:
    跳过该帧
```

## 5. 分析方法

### 5.1 三区段划分

| 区域 | X 范围 | 含义 |
|------|--------|------|
| 左臂 (Left) | X < 500 | 果蝇进入左臂 |
| 犹豫区 (Hesitation) | 500 ≤ X ≤ 700 | T 型交汇处，果蝇正在选择 |
| 右臂 (Right) | X > 700 | 果蝇进入右臂 |

### 5.2 两个核心比率

**Ratio (1) — 三区段时间分配**：
```
左% : 犹豫% : 右% = 左臂帧数/总帧数 : 犹豫区帧数/总帧数 : 右臂帧数/总帧数
```

**Ratio (2) — 左/右偏好（排除犹豫区）**：
```
L/(L+R) = 左臂帧数 / (左臂帧数 + 右臂帧数)
```

- `L/(L+R) > 0.5` = 偏好左臂
- `L/(L+R) < 0.5` = 偏好右臂
- `L/(L+R) ≈ 0.5` = 无偏好

### 5.3 统计量

- **SEM (标准误)**：`SD / √n`，衡量组内个体间变异。n=1 时 SEM=0
- **分析范围**：全 T 型迷宫（含竖杆和横杠），不裁剪 Y 轴
- **视频校正**：20%、25% EtOH 及 5% 组新增个体自动左移 100px

## 6. 脚本说明

### 6.1 主要分析脚本

| 脚本 | 用途 | 输出目录 | 何时使用 |
|------|------|---------|---------|
| **`maze_analysis_v5.py`** | **位置偏好分析**（当前推荐） | `analysis_output_v{N}/` | 每次更新数据后运行 |
| `maze_orientation_analysis.py` | 朝向分析 | `orientation_analysis_v{N}/` | 需要分析果蝇朝向时 |
| `plot_trajectories.py` | 全局轨迹可视化 | 项目根目录 | 需要总览所有轨迹时 |

### 6.2 历史版本（仅供了解方法演进，日常使用 v5 即可）

| 版本 | 脚本 | 方法差异 |
|------|------|---------|
| v1 | `maze_analysis.py` | 两区段，Y 裁剪，X=570 |
| v2 | `maze_analysis_v2.py` | 两区段，Y 裁剪，X=600，含竖杆过滤 |
| v3 | `maze_analysis_v3.py` | 三区段，Y 裁剪 |
| v4 | `maze_analysis_v4.py` | 三区段，无 Y 裁剪（方法论与 v5 相同） |
| **v5** | `maze_analysis_v5.py` | 三区段，无 Y 裁剪，**自动版本号** |

### 6.3 自动版本管理

每次运行 `maze_analysis_v5.py` 时，脚本会：
1. 扫描已有 `analysis_output_v*` 目录
2. 提取版本号，取最大值 +1
3. 创建新目录，输出结果

**无需手动修改代码**。历史结果永不覆盖。

### 6.4 偏移校正说明

部分视频在录制时画面发生了偏移。脚本通过 `X_OFFSETS` 字典自动校正：

```python
X_OFFSETS = {
    '20%EtOH_air_WT': -100,       # 20% EtOH 左移 100px
    '25%EtOH_air_WT': -100,       # 25% EtOH 左移 100px
    '5%ETOH-air-WT -3DLC': -100,  # 5% 新增个体左移 100px
}
```

如需为新文件添加偏移，编辑此字典即可。

## 7. 输出文件说明

每次运行 `maze_analysis_v5.py` 产生 **5 个文件**：

### step1_heatmaps.png
**2×3 位置热力图**。每个子图对应一组，颜色越亮表示果蝇在该位置停留的帧数越多。

- 金色竖线：X=500（左臂/犹豫区边界）
- 橙红竖线：X=700（犹豫区/右臂边界）
- 左上角标注：三区段时间比例和 L/(L+R)

> 用于：快速判断各组果蝇的活动热点区域

### step2_trajectories_review.png
**2×3 轨迹散点图**。每个子图叠加了同组所有果蝇的逐帧位置。

- 蓝色底色 = 左臂区域
- 黄色底色 = 犹豫区
- 红色底色 = 右臂区域
- 不同颜色的散点 = 不同个体

> 用于：检查轨迹质量，发现异常运动模式

### step3_group_preference.png
**组级偏好对比图**（两张子图）。

- **左图**：堆叠柱状图，显示各组在左臂/犹豫区/右臂的时间百分比
- **右图**：L/(L+R) 柱状图，柱高 > 0.5 = 左偏（蓝），< 0.5 = 右偏（红），含 SEM 误差线

> 用于：课程报告的核心图表，展示乙醇剂量-反应关系

### step3_per_file_preference.png
**个体偏好详情图**（两张水平柱状图）。

- **左图**：每只果蝇的三区段时间分配
- **右图**：每只果蝇的 L/(L+R)，按组别分隔（黑色横线 = 组边界）

> 用于：检查个体间差异，发现异常个体

### statistics_report.txt
**纯文本统计报告**，包含：
- 文件分组信息
- 组级三区段帧数和比例
- 组级 L/(L+R) 及 SEM
- 每个个体的详细统计

> 用于：提取准确数字写报告，或导入其他软件进一步分析

## 8. 当前结果 (v10)

*运行时间: 2026/07/09 | 数据: 13 只果蝇，6 个处理组*

### 8.1 L/(L+R) 比率总览

| 分组 | 个体数 | 总帧数 | L/(L+R) | SEM | 偏好 |
|------|:----:|--------|:-------:|:---:|------|
| Air (Ctrl) | 5 | 57,582 | **0.5555** | 0.0577 | 轻微左偏 |
| 5% EtOH | 3 | 52,378 | **0.4990** | 0.1852 | 均值中性，个体差异极大 |
| 10% EtOH | 1 | 7,649 | **0.5848** | — | 左偏 |
| 15% EtOH | 1 | 3,518 | **0.7427** | — | 强左偏 |
| 20% EtOH | 2 | 15,991 | **0.3952** | 0.0507 | 右偏 |
| 25% EtOH | 1 | 11,031 | **0.2378** | — | 强右偏 |

### 8.2 三区段时间分配

| 分组 | 左臂% | 犹豫区% | 右臂% |
|------|:-----:|:------:|:-----:|
| Air (Ctrl) | 38.5 | 30.7 | 30.8 |
| 5% EtOH | 34.4 | 31.0 | 34.6 |
| 10% EtOH | 35.7 | 38.9 | 25.3 |
| 15% EtOH | 49.1 | 33.8 | 17.0 |
| 20% EtOH | 22.6 | 42.9 | 34.6 |
| 25% EtOH | 14.5 | 39.0 | 46.5 |

### 8.3 主要发现

1. **非线性剂量-反应关系**：低浓度乙醇 (10-15%) 增强左臂偏好，高浓度 (20-25%) 翻转为右臂偏好，25% 右偏最强
2. **5% EtOH 组个体差异最大**（SEM=0.185）：三只果蝇分别表现为强左偏 (0.76)、中性 (0.56)、强右偏 (0.12)，提示该浓度可能处于行为转换的临界点
3. **对照组稳定性**：5 只 Air 组果蝇 L/(L+R) 均值为 0.5555，SEM 仅 0.0577，表明无乙醇条件下果蝇有轻微但一致的左偏倾向
4. **样本量不足**：10%、15%、25% 组各仅 1 只个体，结论需谨慎

## 9. 如何添加新数据

### 步骤 1：放入文件

将新的 `.h5` 文件复制到项目根目录 `Ttrap/`。

### 步骤 2：更新分组

编辑 `maze_analysis_v5.py`，在 `FILE_GROUPS` 中添加新文件的匹配前缀：

```python
FILE_GROUPS = {
    ...
    '30% EtOH':   ['30%EtOH_air_WT'],   # 新增
    'Air (Ctrl)': [...],
}
```

并在 `GROUP_ORDER` 中插入新组名。

### 步骤 3（如需要）：添加偏移校正

如果新视频有画面偏移，在 `X_OFFSETS` 中添加：

```python
X_OFFSETS = {
    ...
    '30%EtOH_air_WT': -100,  # 新增，根据需要调整偏移量
}
```

- 正数 = 右移，负数 = 左移
- 前缀必须能唯一匹配目标文件名开头

### 步骤 4：运行

```bash
python maze_analysis_v5.py
```

结果自动输出到新版本目录。
